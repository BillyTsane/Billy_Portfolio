Time Tracking Project

Project Description

This project involved developing a Time tracking Software.
The Workers or the Employees should be able to check in and check out when ever they start or end Working.
Others Possibilities are for the Workers to be able to make Holidays requests, see their Colleagugues Info Details, for Example if they are online or not.
